# leave

#### 介绍
今日校园模拟软件，

软件源码仅提供学习，

用于学习安卓布局，activity事件等。

今日起不再提供发行版，

仅供学习。

有bug可以在下面留言，

如果对你有帮助请请点一个star


#### 软件架构
软件架构说明


#### 安装教程

1.  Android studio/idea克隆本项目
2.  按需修改
3.  打包

#### 使用说明

1.  到统计页面找到发行版
2.  安装需求下载三级审核版和非三级审核版
3.  安装app
